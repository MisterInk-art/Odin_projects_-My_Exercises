Types of Lists

Unordered Lists

Unordered lists can be declared using this code:

                
<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>
Example:
  • Unordered lists don't have numbers
  • List element: <ul>
  • List item: <li>

Ordered Lists

Ordered lists can be declared using this code:

                
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>
Example:
  1. Ordered lists display numbers
  2. List element: <ol>
  3. List item: <li>
