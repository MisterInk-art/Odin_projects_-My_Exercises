Attributes explained

href="https://www.theodinproject.com/about"
his is the most crucial part. It specifies the URL that the link points to.
target="_blank"
This attribute tells the browser to open the linked page in a new tab or window. This is important for user experience so they don't navigate away from your site.
rel="noopener noreferrer"
Phis is a very important security measure, especially when using target="_blank". It prevents the new tab from being able to manipulate the original page through the window.opener property. This is a critical step to avoid potential security vulnerabilities. It's almost always best practice to include this when using target="_blank".

Absolute and relative links

Absolute links

Absolute links are used to link to resources on different websites. They contain the full URL, including the protocol (scheme), domain name, and any path to a specific page or resource. The general format is: scheme://domain/path. Both the scheme (e.g., https://) and the domain (e.g., www.example.com) are essential components of an absolute link.

For example, https://www.theodinproject.com/about is an absolute link to the "About" page on The Odin Project website.

Realtive links

Relative links are used to link to other pages on the same website. They specify the path to the target resource relative to the current page's location. Because they link within the same domain, the domain name is not included. Relative links can also be used to link to different sections of the same page.

For example: <a href="contact.html">Contact us</a> links to the "contact.html" page within the same website.
